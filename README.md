# PowerShell Scripts

PowerShell scripts for IT administrators, Microsoft 365, Azure, Active Directory, and Windows Server.

## About This Repository

This repository contains PowerShell scripts that can help IT administrators with common administration, reporting, checking, and automation tasks.

The scripts are organized by category to make them easier to find and use.

## Categories

* Microsoft 365
* Exchange Online
* Microsoft Entra ID
* Active Directory
* Azure
* Windows Server

## Important Warning

Always review any PowerShell script before running it.

Test scripts in a lab or test environment before using them in production.

These scripts may need to be modified to match your own environment.
